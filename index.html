Finished Projects

BioPank Modeling & Analysis (iGEM)

Kinetic model to describe Antimicrobial Peptides production based on B. subtilis growth rate and other parameters. This model contains first-order ODEs and applies a Global Sensitivity Analysis to find critical parameters from the model. Using odeint and plotly packages (Python Packages) for model simulation and R for data visualization from sensitivity analysis results.

Carpincho Toehold Designer (iGEM)

The design of toehold switches requires an in silico knowledge of the thermodynamic features that can influence their application. I developed the Carpincho Toehold Designer for SynFronteras iGEM team (UNILA_LatAm), the applicaton uses NUPACK Design and Analysis algorithms (Python package).
